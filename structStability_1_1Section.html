Detailed Description

Represents a single hydrostatic data point.

Represents a single sectional hydrostatic data point.

Typically stores data for a given draft:

  • Waterplane area
  • First moment of area (KB)
  • etc.

This can be used for interpolating hydrostatic properties (area, moment, etc.) at a given draft.
